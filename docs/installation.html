Installation

There are a few different ways to install Proem for use. The first and easiest is to simply download the phar archive and include it where you need it, the second is to download the source and place the lib directory onto your include_path.

Which one you choose really depends on what your going to do with Proem.

Dependencies

Proem was developed from the ground up to take advantage of some of the newer features that are only available in PHP5.4. As such it requires PHP5.4 to run.

Other than the PHP5.4 dependency there are currently no other hard requirements to use Proem.

Having said that however, if you plan on contributing to the Proem project, plan on installing Proem from source or plan on executing Proem's test suite we recommend you have Git installed as well as the Phake build tool.

Enable rewrite rules

In order for all the necessary requests to your server to be processed by Proem you need to rewrite these requests to your bootstrap.php file.

Using the Apache HHTP Server you can do this by defining a simple rewrite rule. 

    RewriteEngine on
    RewriteCond %{REQUEST_FILENAME} !-f
    RewriteCond %{REQUEST_FILENAME} !-d
    RewriteRule !\.(js|ico|gif|jpg|png|css)$ /index.php

Using the phar archive

The phar archive route is by far the easiest to get up and running with. Simply download the latest phar archive and include it into your bootstrap.php file for use. 

    include 'phar://proem.phar';

    (new Proem)->init();

The phar archive takes care of adding Proem to the Autoloader so everything is just ready to go.

Installing the source

You can get Proem's source by either downloading the latest package from here or by cloning the repository using git.

git clone git://github.com/proem/proem.git

Once you have the source code, to make use of it, all you need do is put the lib directory on your include_path.

Edit your php.ini file to read similar to: 

    include_path = ".:/usr/share/php:/path/to/proem/lib"

Even though Proem is now on your include path you still need to manually add the name space to the Autoloader within your bootstrap.php file. 

    require_once 'Proem/Autoloader.php';

    use Proem\Autoloader;

    (new AutoLoader)
      ->registerNamespace('Proem', '/path/to/proem/lib')
      ->register();

More information on can be found in the Autoloader docs.

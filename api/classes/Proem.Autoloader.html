Load a class

load(string $class) : \Proem\Api\Autoloader

This load mechanism is not only responsible for locating and including the file where a class is defined, but is also responsible for implementing Proem's cascading file system. This is achieved by suffixing \Api onto the Proem part of any namespace starting with Proem, including the class from within the Proem\Api namespace, and then aliasing that class back to Proem (without the \Api suffix)

link http://proemframework.org/docs/cascading-namespace.html
